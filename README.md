## Problem Statement
- Fragmented Apps: Users juggle multiple travel platforms separately. 
- Transit Confusion: Passengers unsure which bus route. 
- Safety Gaps: Limited emergency features in travel apps. 
- Inefficiency: No unified platform for complete journey. 
- User Frustration: Time-consuming planning, poor user experience. 
- Safety Risk: Unfamiliar areas lack contextual guidance features.
## Objective of the Project
- Unified Travel Platform: Single app for Kerala mobility. 
- Seamless Transit Helper: Bus routing, tracking, boarding guidance. 
- Enhanced Safety: One-tap SOS, emergency contact integration. 
- Contextual Awareness: Weather, heat, shade-aware route suggestions. 
- Societal Impact: Safe, accessible commuting for all Keralites. 
- Reduce Fragmentation: Eliminate need for multiple travel apps. 
- Empower Unfamiliar Passengers: Clear instructions for new routes.
