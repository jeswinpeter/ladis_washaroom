### Who can contribute?
Anybody with knowledge in the tech stack, good understanding of the project and/or a good ai subscription.
### How to contribute?
* Fork the repo
* do the work raise a pull request
* patiently wait for us to respond (we are not busy, we are just trying to keep our aura 😎)
# Important formalities
* Raise an issue if you find any.
* !! Any work done should be well documented !! -> propper comments should be added for the other contributors understanding
## About raising issues
* Try to add proper tags to the issue
* The issue should be well described
* The expected fix or requirements should be mentioned along with the issue.
